# Silent Disengagement Detection

Early detection of silent student disengagement using behavioral patterns and explainable AI.

## Problem
Students disengage weeks before failure, but education systems detect issues too late.

## Solution
Analyze behavioral engagement timelines to identify early disengagement signals and generate ethical, human-centered intervention suggestions.

## Key Principles
- No grading or ranking
- Behavior-based detection
- Explainable insights
- Privacy-first design

## Status
Core detection logic implemented.
